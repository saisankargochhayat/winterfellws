MACHINE LEARNING

The works We are proud of

DOOT 1.0

This project aims at converting gesture to speech using Leap Motion Device.Based on Machine Learning,the Leap device has been trained and does live predictions. ML Library "Scikit-Learn" and its classifiers have been used to build this.

CLEAN CLAN

Echoing Swachh Bharat Aviyan,this project gamifies "swachhta" across the country.People can contribute as well as participate in this web app,which further awards them for their actions.

1TOUCH

This projects is a "Liquid level control system", used for monitoring liquid levels in tanks and reservoirs. Usually, these kind of systems provides visual multi-level as well as continuous level indication. Audio visual alarms at desired levels and automatic control of pumps based on user’s requirements can be included in this managementn system using Arduino

INDO-DRIVE

It is a self-driving bot with lane and obstacle detection. It too uses Machine learning for its development

MANUS

We built a robotic hand which is capable of performing pick and place actions with a degree of freedom 6 and good precision. At present we are controlling it with a USB mouse and our target is to give it a anthropomorphic behavior i.e. a hand which can be controlled by human gestures.
